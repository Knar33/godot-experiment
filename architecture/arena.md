# Arena Architecture

## Scene Structure

`scenes/Arena.tscn` is a standalone scene instanced into `Game.tscn`.

```
Arena (Node3D) [scripts/Arena.cs]
├── Floor (StaticBody3D)
│   ├── FloorMesh (MeshInstance3D) — PlaneMesh 50x50 with grid shader
│   └── FloorCollision (CollisionShape3D) — BoxShape3D 52x1x52, centered at y=-0.5
├── BoundaryWall_0..31 (StaticBody3D) — generated at runtime by Arena.cs
└── SpawnPoint_0..11 (Marker3D) — generated at runtime by Arena.cs
```

## Floor Rendering

The floor uses a `PlaneMesh` (50x50) with a custom spatial shader that:

- Converts UV coordinates to world positions: `(UV - 0.5) * 50.0`
- Draws a grid pattern (2-unit spacing) for movement readability
- Adds a blue emissive glow at the center for orientation
- Adds a red emissive glow ring at the arena edge to mark the boundary
- Discards fragments outside the arena radius (25 units) to create a circular shape from the square mesh

The floor collision is a large flat `BoxShape3D` (52x1x52) centered below y=0 so its top face aligns with the mesh surface.

## Boundary Walls

Generated procedurally by `Arena.cs` in `_Ready()`. 32 invisible `StaticBody3D` segments with `BoxShape3D` colliders form a ring just outside the arena radius. Each segment is:

- Positioned at radius 25.25 (arena radius + half wall thickness)
- Rotated tangent to the circle
- Slightly oversized (1.1x arc length) to prevent gaps
- 4 units tall, 0.5 units thick
- No mesh — the visual boundary is the shader's edge glow

## Spawn Points

12 `Marker3D` nodes generated by `Arena.cs` at radius 28 (3 units outside the arena boundary), evenly distributed every 30 degrees. Positions stored in `Arena.SpawnPoints` array for the enemy spawner to use.

`Arena.GetRandomSpawnPoint()` returns a random spawn position.

## Key Constants

| Property | Default | Configurable via |
|----------|---------|------------------|
| Arena radius | 25 units | `[Export] Radius` |
| Boundary segments | 32 | `[Export] BoundarySegments` |
| Wall height | 4 units | `[Export] WallHeight` |
| Spawn point count | 12 | `[Export] SpawnPointCount` |
| Spawn point radius | 28 units | `[Export] SpawnPointRadius` |
| Grid spacing | 2 units | Shader uniform |
